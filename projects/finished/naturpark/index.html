Naturpark Fichtelgebirge

Redesign of the already existing website for the Fichtelgebirge National Park.

For this project we worked in a team of three people. Our goal was to give the already existing website a complete makeover with better structure, new photos, content and a consinstend design. We started with wireframes, personas and user journeys to get an outline for our project. We definded fitting typography and a colour concept.

The prototype was realized in Figma. Later the website was realized with html and css. The finished project can be found here.
